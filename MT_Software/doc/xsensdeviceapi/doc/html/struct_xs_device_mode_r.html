Detailed Description

A structure for storing device modes.

Stores device modes using an update rate.

Note
MT Mk4 devices are better configured using XsOutputConfiguration, this object is supplied for legacy support only.

Compute the period and skip factor from the stored update rate.

This function computes the period and skipFactor fields from the stored m_updateRate field. The maximum error in the update rate is approximately 0.4%, which occurs at 510Hz (= actually 512 Hz). In general, the higher rates are harder to set up exactly.
