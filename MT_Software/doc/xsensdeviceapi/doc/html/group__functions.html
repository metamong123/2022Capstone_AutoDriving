◆ xdaEnableNetworkScanning()

void xdaEnableNetworkScanning ( )

Enable network scanning functionality.

When calling this function, the network scanning functionality will be enabled. As a result, XsScanner will include network devices in its scanPorts results. Network scannning is done in a background thread and may affect system performance. It is recommended to only enable it when needed.

Any call to xdaEnableNetworkScanning should be mirrored by a call to xdaDisableNetworkScanning when done with the scanning as these calls are internally reference counted.

See also
xdaDisableNetworkScanning

◆ xdaGetLogger()

Journaller* xdaGetLogger ( )

Rreturn the journaller use for logging.

This function can be used to share the same log file between different binary objects. For example calling xdaSetLogger(xdaGetLogger()); will merge all xkf*.log files into the xda log file

Returns
A pointer to the current journaller.

void xdaReadMtbMetaData ( struct XsByteArray *  metadata,
const struct XsString *  mtbFileName 
)

Read meta data from an mtb file.

This function will read the meta data from file mtbFileName if it is present. If the file was recorded with a Bodypack with firmware version 1.5.2 or higher this will succeed, otherwise an empty XsByteArray will be returned.

Parameters
metadataWill be filled with the requested meta-data
mtbFileNameThe filename of the mtb file to read from

◆ xdaSetLogger()

void xdaSetLogger ( Journaller *  journaller)

Set the journaller to use for logging.

This function switches the logging to the desired journaller. Existing contents will be moved to the new journaller, erasing the old file.

Parameters
journallerThe new Journaller to use, when set to nullptr, xda will stop debug logging completely
