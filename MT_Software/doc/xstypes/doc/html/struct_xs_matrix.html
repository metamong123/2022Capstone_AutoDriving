◆ stride()

XsSize XsMatrix::stride ( ) const
inline

Return the stride of the matrix.

The stride of a matrix is for internal administration. It defines the number of items in a row in the data buffer. This is always greater than or equal to the number of columns. Especially for matrices that reference a part of another matrix this may differ from the cols() value.

Returns
The stride of the matrix.

◆ swap()

void XsMatrix::swap ( XsMatrix &  b)
inline

Swap the contents of b with this.

This function swaps the internal buffers so no actual data is moved around. For unmanaged data an elementwise swap is done, but only if the matrices are the same size.

Parameters
