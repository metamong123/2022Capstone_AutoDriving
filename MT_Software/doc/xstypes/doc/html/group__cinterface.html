Detailed Description

This section contains all the pure C interface functions and constants.

The C interface is supplied for portability purposes. It prevents the need for compiler-specific libraries, except on completely different platforms (ie. Windows vs Linux).

If you have a C++ environment, there is no need to use the C interface and it is even recommended to ONLY use the C++ wrappers. The C++ wrappers provide safe object lifetime management, which you will have to do yourself if you're only using pure C (this mostly means calling the _construct and _destruct functions yourself).

Most of the C++ functions are simple wrappers around the pure C calls and to prevent duplicate documentation and duplicate documentation maintenance, a lot of the wrapper function documentation refers to these C interface functions.
