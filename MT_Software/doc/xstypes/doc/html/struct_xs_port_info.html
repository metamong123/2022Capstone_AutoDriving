Parameters
portNrThe number of the COM port
baudRateThe baud rate to configure for the port, for scanning XBR_Invalid may be used to scan all known baud rates
linesOptionsThe options for the hardware flow control lines
Note
Numbered COM ports are only available on Windows platforms.
