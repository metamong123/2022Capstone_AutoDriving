◆ m_valid

uint8_t XsTimeInfo::m_valid

Validity indicator.

When received: bit (0) - UTC Date is valid; bit (1) - UTC Time of Day is valid; bit (2) - UTC Time of Day has been fully resolved (i.e. no seconds uncertainty).
