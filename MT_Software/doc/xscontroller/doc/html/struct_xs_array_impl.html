Detailed Description

template<typename T, XsArrayDescriptor const & D, typename I>
struct XsArrayImpl< T, D, I >

A C++ wrapper for XsArray, this is a template class for the actual specialized classes.

Template Parameters
TThe type of the contained values
DThe descriptor to use for this specific array implementation. This must be statically allocated and its lifetime must encompass the lifetime of objects that use it.
IThe class that inherits from the XsArrayImpl. Some functions (such as the streaming operator) require the inheriting type to be returned for proper functionality.
