Detailed Description

Internal administration for contained data of XsDataPacket class.

This is only the part that can be stored in an XsMessage, so no TOA and computed packet IDs

MapType::const_iterator DataPacketPrivate::find ( XsDataIdentifier  id) const

Find the item matching id.

This function will return an iterator to the item that matches id or end() if it could not find it. The function uses a loose comparison to ensure entry ambiguity. As a result, the returned iterator may have a slightly different id than the supplied one.

◆ merge()

void DataPacketPrivate::merge ( DataPacketPrivate const &  other,
bool  overwrite 
)

Merge other into this.

The function will copy all items from other into this. Existing items will only be overwritten if overwrite is set to true. This function does not detect conflicts between closely related but IDs (such as frame range and packet counter). This is left to the caller.

Parameters
otherThe other data to merge into this packet
overwriteWhen set to true, the contents of other will overwrite existing data. When set to false, only new items will be added.
