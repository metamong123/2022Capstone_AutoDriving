Detailed Description

template<typename T>
class xsens::GenericMatrix< T >

Generic matrix class.

This class can store items of the given type. Items in the list can be accessed through the [] operator or the get() function.

Do NOT use any item type that requires a constructor to work correctly. Pointers to these objects can work though.
