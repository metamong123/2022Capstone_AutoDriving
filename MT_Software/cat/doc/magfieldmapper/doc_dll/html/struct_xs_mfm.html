Detailed Description

High level Magnetic Field Mapper (MFM) management class.

Most functions return a boolean indicating success, a null-object, empty lists or nothing at all. In some cases more in-depth knowledge of the error is required (

See also
loadInputFile). Therefore they return an XsResultValue
Note
This object cannot be copied. The copy constructor has been disabled.
